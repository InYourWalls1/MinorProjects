IT:
Esecuzione del programma:
*al primo utilizzo, o quando si sposta il programma in una nuova cartella, è possibile che l'icona del file "Dice" (o Dice.lnk) scompaia. per ripristinarla basta eseguire il file "setup_link.bat" (come tutti i file, basta fare click su di esso per due volte affinché si apra).
*è possibile eseguire il programma sia cliccando il file .py sia cliccando il file "Dice". Quest'ultimo è utile se si vuole avere il programma nel menù start o nella taskbar.

Funzionalità:
*le opzioni da 1. a 7. "tirano un dado" con il rispettivo numero di facce (ad esempio l'opzione 2. tira un D6, ovvero un dado a sei facce).
*le opzioni da 8. a 14. sono simili, tuttavia permettono di scegliere se dare un vantaggio (ADV) o uno svantaggio (DIS) a un dado. Se un dado è svantaggiato, il programma tirerà due volte il dado e mostrerà il risultato minore; similmente, se un dado è avvantaggiato, accadrà la stessa cosa, solo che il programma mostrerà il risultato maggiore.
*l'opzione 15. e l'opzione 16. sono simili alle precedenti, con l'eccezione che è possibile scegliere un qualsiasi numero di facce per il dado da tirare. La differenza fra le due sta nel fatto che l'opzione 15. non permette di scegliere uno svantaggio o un vantaggio, mentre l'opzione 16. lo consente.
*l'opzione 17. è utile se si desidera tirare tanti dadi in successione. Vi sono tre "comandi" che è possibile dare al programma in questa modalità:
1. advroll <numero> tira un dado del numero di facce specificato con vantaggio
2. disroll <numero> tira un dado del numero di facce specificato con svantaggio
3. roll <numero>: tira un dado del numero di facce specificato senza vantaggio né svantaggio
i comandi possono essere inseriti in successione, avendo la possibilità di creare comandi complessi come:
"advroll 20 disroll 90 roll 15", che tirerà un dado vantaggioso da 20, un dado svantaggioso da 90 e un dado normale da 15.
*infine l'opzione 0. permette di uscire dal programma.

EN:
Executing the program:
*when first running it, or when moving it to a new folder, the icon of the "Dice" file (or Dice.lnk) might disappear. to restore it simply click "setup_link.bat".
*it's possible to execute the program both by clicking the file .py and by cliking on "Dice" link. the latter is useful if you wish to add the program to the taskbar or on the start menu.

Features:
*options from 1. to 7. "roll a dice" with the respective number of faces (for example the option 2. rolls a D6, which is a 6-sided dice).
*options from 8. to 14. are very similiar, though with them it's possible to choose an advantage (ADV) or a disadvantage (DIS) when rolling. If a dice has disadvantage, the program will roll two dice and show the dice with the smallest result; similiarly, a dice with advantage will do the same thing, just in reverse.
*option 15. and 16. are similiar to the previous ones, with the exeption that it's possible to choose the number of faces to give to the dice. The difference between the two is that while 15. doesn't allow choosing an advantage or a disadvantage, 16. does.
*option 17. is useful if you want to roll many dices in succession. There are  three commands that you can give to the program in this mode:
1. advroll <number> rolls a dice with the specified number of faces with advantage
2. disroll <number> rolls a dice with the specified number of faces with disadvantage
3. roll <number>: rolls a dice with the specified number of faces with neither advantage nor disadvantage
commands can be inserted one after the other, having the possibility to create complex commands such as:
"advroll 20 disroll 90 roll 15", which will throw a D20 with advantage, a D90 with disadvantage and a normal D15.
*finally, option 0. allows to exit from the program.
